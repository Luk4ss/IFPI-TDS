Bem-vindo! Essa é minha primeira aplicação construída com o framework angular.

SOBRE ESSA APLICAÇÃO

Essa página foi construída parar colocar os meus conhecimentos em angular em prática. Aprendi muito com o curso da COD3R "Angular Essencial" e também com o curso da Udemy "Angular The complete Guide - 2022".

Também aprendi muito com o curso de Java da Udemy proporcionado pelo professor Dr. Nelio Alves e por isso o sistema back-end foi construído em Java utilizando o Spring. Utilizei o JPA para realizar a interção com o banco de dados.

O sistema conta com duas entidades: Product e Pet. Por enquanto, está sendo utilizando o banco de dados em memória H2, mas pretendo implementar para um banco de dados como postegres/MysQL.
